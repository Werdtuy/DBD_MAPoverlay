# Maps

Put Dead by Daylight map overlay images here.

Supported formats: `.png`, `.webp`, `.gif`.

Animated GIF and animated WEBP files are supported. Sidecar metadata can be added as `<image-name>.<ext>.json`.

Hens callout maps are downloaded from https://hens333.com/callouts. The source page credits the images to Lethia and identifies the page as Zexov's modified version of the original build by Broosley and Evo from Hens' Discord.
